# LabVeilleTech

Blog de veille technologique sur l'univers des **indie hackers**.

## Stack

- Hugo + PaperMod
- GitHub Pages
- Contenu en Markdown

## Commandes

```bash
# Dev avec drafts
hugo server -D

# Build production
hugo --minify

# Nouveau post
hugo new articles/mon-article.md
```

## Structure

```
content/
├── articles/    # Articles de veille
└── about.md     # Page À propos
```
